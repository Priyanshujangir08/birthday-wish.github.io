# birthday-wish.github.io

A simple birthday greeting web page built as a GitHub Pages site.

## Overview

This repository contains a small interactive celebration page that plays music and renders a heartfelt birthday wish. The page includes animated visuals, a countdown-style message, and an optional canvas interaction.

## Files

- `index.html` - main page markup and inline page script
- `aud.mp3` - background audio played automatically on load
- `file/default.css` - styles for the page
- `file/functions.js` - helper JavaScript used by the page
- `file/love.js` - animation and canvas logic
- `file/jquery.min.js` - jQuery dependency
- `file/jscex*.js` - JSCExtension scripts used by the animation logic
- `img.png`, `img2.png` - supporting images used by the page

## Usage

1. Open `index.html` in a browser.
2. Allow audio playback if prompted.
3. Click the canvas to interact with the animation.

## Deployment

This repository is ready to host as a GitHub Pages site from the `main` branch.

## Notes

- The page is designed to work best in modern browsers such as Chrome and Firefox.
- If auto-play is blocked, use the play controls or click the page to start audio.
